# Base CRM Web

Frontend web application cho Base CRM được xây dựng với React + Vite + Tailwind CSS.

## Cài đặt

```bash
npm install
```

## Development

```bash
npm run dev
```

Ứng dụng sẽ chạy trên `http://localhost:3000`

## Environment Variables

Tạo file `.env`:

```
VITE_API_BASE_URL=http://localhost:3001
```

## Build

```bash
npm run build
```

Output sẽ nằm trong thư mục `dist/`

## Preview Production Build

```bash
npm run preview
```
